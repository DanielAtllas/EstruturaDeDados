Tipos Abstratos de Dados - Estrutura de Dados

Existem situações em que os tipos de dados básicos int, float, double e char não são suficientes para representar os dados que precisamos manipular em nossos programas. Muitas vezes estes dados são Estruturas Complexas que são compostas por outros dados e muitas vezes com várias ocorrências de um dado que faz parte de outro dado.

Estruturas complexas são aquelas que podem armazenar mais de um valor, mais de um elemento de um determinado tipo de dado que pode variar dependendo do problema. Ou seja, será como um bloco que irá conter N variáveis de N tipos.

Os vetores e matrizes podem armazenar mais de um valor, mas são limitados a um único tipo de dado para seus elementos, o que limita muito seu uso para estas situações. O ideal seria termos a possibilidade de criar novos tipos de dados, os chamados tipos abstratos de dados , que possam ser definidos de acordo com o problema e que permitam o uso de mais de um tipo de dado.

A oartir dos tipos básicos da linguagem C (int, char, float e double), é possível criar novs tipos de dados compostos por membros com diferentes tipos de dados, podendo ser compostos por dados de tipo básico ou ainda por dados de outros tipos abstratos, superando assim a limitação dos vetores e matrizes.

Veja o exemplo abaixo, com um tipo Aluno possuindo variáveis matricula, notas e média:

    struct aluno{
        int Matricula
        float Nota[3], Media;
    };
    
    Perceba que essa é a definição de criação de uma estrutura de dados, dando um comando struct seguido pelo nome da estrutura. Após isso, basta inserir as vaiáveis que irão compor 
    esta estrutura e seus tipos.
    
    Após a criação, deve ser feita a declaração da estrutura dentro do main como no exemplo a seguir (Além das atribuições de valores):
    
    int main(){
        struct aluno Aluno1
        
        Aluno1.Matricula=18101381;
        Aluno1.Nota[0]=7,5;
        Aluno1.Nota[1]=5,2;
        Aluno1.Nota[2]=8,4;
        Aluno.Media=(Aluno1.Nota[0] + Aluno1.Nota[1] + Aluno1.Nota[2])/3;
        
        printf(" Matricula: %d",Aluno1.Matricula);
        printf("\n Média: %.2f",Aluno1.Media);
        
        return 0;
    }
    
    Peceba que é feito a declararação a variável Aluno1 do tipo struct aluno, e para que sejam acessados os valores da struct, é passado o nome da variável Aluno1 seguido por um 
    ponto e o nome do valor da struct.
    
    Assim como, também é possível inserir em uma variável do tipo struct o valor de outra variável do mesmo tipo, por exemplo:
    
    struct aluno Aluno1, Aluno2;
    Aluno2 = Aluno1;
  • Estruturas Aninhadas

    • Estruturas podem ser definidas a partir de outras estruturas já definidas anteriormente. Desta forma, podemos ter estruturas aninhadas, o que é uma forma de definição de tipos abstartatos que promove o reuso e facilita a manutenção dos programas. Podemos definir uma estrutura contendo informações usadas em várias partes do programa em um único ponto e utilizar esta estrutura na definição de outras estruturas que precisam das mesmas informações. Resumindo, é possível inserir em uma struct os dados de uma outra struct, como no exemplo a seguir:

                typedef struct{
                int dia, ano;
                char mes[10];
                
            }data;
            
            typedef struct{
                int codigo;
                floatpreco;
                Data DataVenda;
            }venda;
            
            Dessa forma, todos os dados da struct Data (Dia,mes,ano) serão armazenados na struct Venda. Para que os dados sejam armazenados, o procedimento é muito semelhante aos mostrados anteriormente:
            
            int main(){
        struct venda Venda
        
        Venda.DataVenda.dia = 25;
        Venda.DataVenda.mes = 05
        Venda.DataVenda.ano = 1999;
        
        return 0;
    }
  • Funções que retornam structs

    • Podemos escrever funções que retornam structs, sendo uma forma de fazer com que uma função retorne mais de um valor, pois uma estrutura pode conter mais de um campo e até mesmo campos definidos como vetores e matrizes. As funções também podem receber parâmetros que sejam do tipos struct. Veja abaixo um exemplo de função que retorna uma struct:

                OBS: Lembre-se que nestes casos, a struct deve sempre vir antes do chamador da função.
            
            Venda TotalVendas(Venda A, Venda B){
                Venda total;
                total.pecas = A.pecas + B.pecas;
                total.preco = A.preco + B.preco;
                
                return total
            }
  • Vetores de Structs

    • É possível também, declarar matrizes e vetores usando structs, desde que tenham sido definidas no programa. O exemplo abaixo ilusra um programa que faz uso de um vetor cujos elementos são do tipo struct:

                typedef struct{
                int dia, ano;
                char mes[10];
            }Data;
            
            int main(){
                Data data[5];
                int i,n;
                
                for(i=0;i menor 5; i++){
                    printf("Informe o valor do dia: ");
                    scanf("%d",&data[i].dia);
                    
                    printf("Informe o valor do mes: ");
                    scanf("%d",&data[i].mes);
                    
                    printf("Informe o valor do ano: ");
                    scanf("%d",&data[i].ano);
                }
            }
